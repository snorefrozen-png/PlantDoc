# PlantDoc Mobile Application

## Description
PlantDoc is an Android mobile application designed to help farmers diagnose crop diseases and pest infestations. Users can submit descriptions and images of affected plants, and the system provides possible diagnoses to assist in early detection and treatment.

---

## Features
- Submit plant symptoms using text descriptions  
- Upload images of affected crops  
- Receive possible disease or pest diagnosis  
- User-friendly mobile interface  
- Supports faster decision-making for farmers
